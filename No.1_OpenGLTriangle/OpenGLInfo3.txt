3.7 渲染上下文

EGL 为支持的客户端 API 提供了一些函数，用来创建并销毁渲染上下
文（rendering contexts）；查询关于渲染上下文的信息；将渲染上下
文绑定到界面（即，当前绘制的界面）。

每个支持的客户端 API 最多


在同一时刻，每个支持的客户端 API 最多有一个上下文关联到一个指
定的线程；（即，上下文是和线程关联的）
在同一时刻，最多有一个上下文和指定的界面绑定；

EGL 实现支持的最小数量的当前线程对于每种支持的客户端 API 来说
是 1。

只有一个 OpenGL 或 OpenGL ES 上下文对应到一个特定的线程，即使
在实现的时候支持 OpenGL 和 OpenGL ES 1.x 和 OpenGL ES 2.x 在同
一个事务中。使用 eglMakeCurrent 来强制限制。
